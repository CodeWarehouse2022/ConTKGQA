# ConTKGQA
## Dataset
Please download the dataset [CronQuestions](https://github.com/apoorvumang/CronKGQA) and save it under the folder "datasets/CronQuestions"
## Requirements
Python version >= 3.9

Pytorch >= 1.10.0
```
pip install -r requirements.txt
```

## Train

```
python train.py
```

## Test

```
python test.py
```
